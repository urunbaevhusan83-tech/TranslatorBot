# TranslatorBot
# Telegram-бот переводчик

Данный проект представляет собой Telegram-бота, предназначенного для
перевода текста на разные языки прямо в мессенджере Telegram.

## Актуальность
Боту не нужно устанавливать отдельные приложения для перевода.
Пользователь получает перевод прямо в чате, что экономит время.

## Функции бота
- Перевод текста на разные языки
- Простое взаимодействие с пользователем
- Работа через команды Telegram

## Используемые технологии (зависимости)
- Python
- Telegram Bot API
- Библиотека telebot, requests

## Инструкция

1. Для начала работы бота нужно ввести команду для запуска (/start)
2. Затем нужно выбрать с какого языка будет рализовываться перевод (их 5) Русский, English, Français, Deutsch, Español.
3. После этого нужно выбрать на какой язык будет осуществляться перевод.
4. Нужно ввести текст который хотите перевести.
5. Бот возвращает переведённый текст пользователю.
